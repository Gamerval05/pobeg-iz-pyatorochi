This package contains some scripts that you can control the door and window. It can be used in many scenes, such as room, or shop and fantasy place. Just few setup then you can animate the door and window.

Features:
- Easy to setup
- 5 scripts for door and window controlling
- Single door, door group and automatic door controller
- Window controller
- 2 scenes: one for houses and one for multiple types of doors test in a small area
- Bonus models for basic house and room
- Open source

Suitable for all platforms.
